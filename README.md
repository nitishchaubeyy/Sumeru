# Sumeru. 🏔️

**Sumeru.** is a modern, scalable, and minimalist *Naam Jap (Chanting) Tracker* designed to help seekers maintain consistency in their spiritual journey.

---

## ✨ Features

### 📱 Progressive Web App (PWA)
- Install Sumeru. on **Android, iOS, or Desktop**
- Full-screen, distraction-free experience
- Native app-like performance

### 🔢 Dual Counting System
- Real-time **Tap-to-Count** with haptic feedback
- Manual entry option for end-of-day updates

### 🌐 Offline-First Architecture
- Works seamlessly with limited or no internet
- Automatically syncs data when back online

### 🎯 Sankalp Tracker
- Set **daily and long-term goals**
- Track progress and stay disciplined

### 👥 Community Mandals
- Create or join groups
- Chant collectively with friends and family

### 🏆 Global Leaderboard
- View top chanters:
  - Daily
  - Weekly
  - Monthly
- Stay motivated through healthy competition

### 📖 Santvani Feed
- Curated spiritual content
- Motivational discourses to stay aligned

---

## 🛠️ Tech Stack

**Frontend**
- Next.js 15 (App Router)
- Tailwind CSS

**Backend**
- Supabase (Authentication + PostgreSQL)

**Standards & Architecture**
- Progressive Web App (PWA)
- Manifest & Metadata API

**State Management & Sync**
- TanStack Query
- IndexedDB (for offline persistence)
